## API Reference
### Endpoints
- `GET /api/status` - Check system status
- `POST /api/data` - Submit new data
- `GET /api/health` - Health check endpoint

### Authentication
Include your API key in the Authorization header:
```
Authorization: Bearer your-api-key
```
